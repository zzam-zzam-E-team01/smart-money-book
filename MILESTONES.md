# 마일스톤: 구현 현황 정리

이 문서는 화면단(HTML/CSS/JS) 기준으로 현재 구현된 기능과 미구현(추가 개발 필요) 기능을 정리합니다.

## 범위
- 화면: `index.html`, `calendar.html`, `account/*`, `board/*`, `login.html`, `signup.html`
- 스크립트: `assets/js/*`
- 데이터 저장: `localStorage` 기반 (샘플 데이터 포함)

## 마일스톤 요약

| 마일스톤 | 구현된 부분 | 구현되지 않은 부분 |
| --- | --- | --- |
| M1. 인증/사용자 관리 | 회원가입/로그인 UI, 입력 검증, `localStorage` 사용자 저장, 로그인 상태 체크, 로그아웃 | 서버 연동, 비밀번호 해시/보안, 세션/토큰 기반 인증, 권한 관리 |
| M2. 일일 가계부(메인) | 월 이동, 월 요약(수입/지출/합계), 일자별 거래 리스트 렌더링, 거래 클릭 시 수정 화면 이동, FAB로 등록 이동 | 서버 데이터 조회/필터, 검색/필터 기능, 실제 삭제/수정 저장 |
| M3. 달력 가계부 | 월 이동/오늘 이동, 월 요약, 날짜별 합계 표시, 날짜 선택 상세 목록 렌더링 | 서버 연동, 고급 필터/검색, 이벤트/거래 CRUD |
| M4. 가계부 등록/수정 | 카테고리/유형 선택 UI, 폼 검증, 신규 거래 `localStorage` 저장, 수정 폼 입력 UI | 서버 저장/수정 API 연동, 수정 시 실제 데이터 업데이트, 삭제 기능 |
| M5. 가계부 통계 | Chart.js 기반 도넛/막대 차트 렌더링(샘플 데이터) | 서버 기반 통계 데이터, 기간/필터 기반 동적 갱신 |
| M6. 게시판 | 목록/등록/수정/상세 화면 UI | 게시판 데이터 로딩/등록/수정/삭제 API 연동 |

## 비고
- 모든 데이터는 현재 정적 샘플 또는 `localStorage`에 저장되며, 서버 API는 주석 예시 수준입니다.
- 실제 서비스 연동 시 각 마일스톤의 “미구현” 항목이 작업 범위가 됩니다.
